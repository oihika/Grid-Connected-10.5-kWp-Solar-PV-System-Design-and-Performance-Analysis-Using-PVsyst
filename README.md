## 📖 Overview

This project presents the complete engineering design and performance analysis of a **10.5 kWp Grid-Connected Solar Photovoltaic System** developed using **PVsyst**.

The project demonstrates the complete engineering workflow including PV array sizing, inverter sizing, electrical design, voltage verification, protection coordination, cable sizing, and system performance evaluation according to industry engineering practices.

---

# 🎯 Project Objectives

- Design a **10.5 kWp Grid-Connected Solar PV System**
- Optimize PV array configuration
- Select an appropriate SMA inverter
- Validate electrical operating conditions
- Evaluate annual energy generation
- Analyze system losses
- Improve overall system efficiency
- Demonstrate professional PV system engineering using PVsyst

---

# ⚡ Problem Statement

Grid-connected photovoltaic systems require careful engineering design to achieve high efficiency and long-term reliability.

Incorrect sizing of PV modules, strings, or inverters may result in:

- Reduced energy production
- Inverter clipping
- Higher electrical losses
- Voltage mismatch
- Poor economic performance

This project addresses these challenges by developing a technically optimized **10.5 kWp PV system** using professional engineering calculations and PVsyst simulation.
